# UDNA — Universal DID-Native Addressing

**Status:** W3C Community Group Incubation  
**Repository:** [github.com/w3c-cg/udna](https://github.com/w3c-cg/udna)

UDNA establishes Decentralized Identifiers as first-class network primitives, enabling identity-native addressing instead of location-based addressing (IP/DNS).
