Welcome to the **Sirraya Protocol Specifications** — the canonical home for the zero-trust agent execution protocol ecosystem.

## What is Sirraya?

Sirraya is a protocol for **secure, verifiable agent execution** in zero-trust environments. It enables agent-to-agent communication, cryptographic trust, decentralized identity, and composable capabilities through a codon-based protocol.

## Structure

| Layer | Description |
|-------|-------------|
| **Core Layer** | Foundational concepts, threat model, trust model |
| **Protocols** | Formal protocol documents (codons, A2A, envelopes) |
